Project readiness notes
=======================

This page is kept as an internal project checklist rather than a main user-guide
page.  For public-facing documentation, start with the use-case gallery,
benchmark gallery, algorithms, API reference, and external/Kaggle gallery.

Before a broad public release, review the API names, build wheels, rerun the
benchmark reports on a pinned machine, and check that the external result pages
state both strengths and limitations honestly.
