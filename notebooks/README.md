# robustcov notebooks

These notebooks are clean, reproducible entry points for users who prefer an interactive workflow.

## Notebooks

- `01_quickstart_robust_distances.ipynb` — fit FastMCD, compute robust distances, and save diagnostic plots.
- `02_small_sample_heavy_tail.ipynb` — compare Cauchy, Student-t, and regularized Tyler scatter in a high-dimensional heavy-tail setting.
- `03_credit_card_fraud_external_template.ipynb` — Kaggle-style fraud workflow; requires external `creditcard.csv`.
- `04_finance_market_stress.ipynb` — finance stress and rolling-window anomaly diagnostics.

External-data notebooks are optional and are not part of tests.
